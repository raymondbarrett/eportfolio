the creation of the boats 94 mod

nhl 94 is a game by EA that allows users to play as their favorite nhl teams

I modded the game to replace the hartford whalers with a local roller hockey team, boats 'n' hossas.

the game is programmed entirely using series of hex codes

Why would anyone make this?

I made this fun little mod mostly for the enjoyment of the people on the team.
It is fun to see how people translate from real-life to the video game.
Thanks to the people who run nhl94.com, the modding community's expertise is what made this possible

Making the roster

Without a doubt the most enjoyable part of this whole process was adding the new roster and uniforms
This is thanks to a tool called NOSE made by forum user wboy
It makes editing all the information super easy, even providing a great GUI to make sure that everything is the way you want it
After some ...creative rankings for each player on the team, the new players were in the game in less than a day.

Custom images

The most time-consuming and frustrating part of the process was the images. Having only 16-bit color on the Genesis and a very limited
memory meant that the developers had to get creative with how they put images in the game.
Essentially, there is hex code that assigns a color id to each pixel and another block of hex code to map colors to color id's.
The center-ice image was fairly easy to edit as detailed in this guide, but the select screen logo caused some headaches
This is because in order to save space, the developers of the game used only half of the hartford whalers logo (which is what I needed to edit) and had code to mirror it instead.
It took me a while to figure out, and my disappointment was immesurable when I realized hartford was the only team with a symmetric logo in the entire game.
After sorting that out, it was time to edit the player pictures on the team. Even though in the original game each team only had pictures for a select number of star players (to save on space)
I wanted to have pictures for everyone on the team. To get around this, I identified the absolute worst teams in the game (other than hartford) and replaced their stars' pictures with my own,
and after modifying the hex code to tell the game where the new pictures were, I was done. After ironing out a few bugs, the game booted up perfectly and ran better than I could've ever hoped for.
If you are looking to try this yourself, please check out the modding forum.
